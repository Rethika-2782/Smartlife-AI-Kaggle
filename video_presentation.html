What is SmartLife AI?
A secure, intelligent smart home ambient controller built on Google ADK
🤖

Multi-Agent System

Orchestrator, Device Agent, and Climate Agent working in concert for specialized domain handling

🔌

MCP Integration

Local Model Context Protocol server exposes strictly typed tools — LLM never touches raw device APIs

🔐

Security-First

PII scrubbing, prompt injection detection, and human-in-the-loop approval for critical operations

System Architecture

1
Security Checkpoint — Scans for PII and injection attacks. Blocks violations immediately.
2
Orchestrator Agent — Main coordinator. Delegates to Device or Climate Agent via AgentTools.
3
MCP Server — Isolated subprocess. All device reads/writes go through strictly defined schemas.
4
Post-Orchestration Router — Detects critical actions and routes to Human Approval Pause.
5
HITL Node — Pauses execution and awaits explicit human confirmation before proceeding.

Security Design Layers

🔍
PII Scrubbing — Regex detects emails and US phone numbers, replacing them with tagged placeholders before any LLM call.
🚫
Prompt Injection Protection — Phrases like "ignore previous instructions" are blocked at checkpoint, short-circuiting the flow entirely.
📋
Structured Audit Logs — Every evaluation produces JSON logs at INFO, WARNING, or CRITICAL levels for full auditability.
⚡
Command Validation — Queries over 300 characters are flagged to prevent resource exhaustion attacks.
